Webhook Reference

Import2 sends import status updates to the webhook URL specified while registering your application via HTTP Post request. Currently, webhook functionality doesn’t re-send import status if your app was unreachable or has responded with an HTTP error.

Webhook URL is called in the following cases:

  • Customer has started a new import
  • Import2 has completed a sample import
  • Import2 has completed a full import
